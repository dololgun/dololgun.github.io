```bash
sftp remote_username@server_ip_or_hostname

-p : custom port
```

명령어 목록을 볼 수 있다.

```
help
```

cd : 리모트의 디렉터리 변경

ls : 리모트 디렉토리 목록

mkdir : 리모트의 폴더를 생성

pwd : 리모트의 현재 디렉터리 위치

lcd : 로컬 디렉터로 패스 변경

lmkdir : 로컬 디렉터리 생성

lpwd : 로컬의 현재 위치

get : 리모트의 파일을 다운로드 한다.

put local [remote] : 파일을 업로드 한다.
